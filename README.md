# CONTA BANCÁRIA

Primeira aplicação desenvolvida em Java. Consiste em um simples projeto de conta bancária que simula operações básicas para fins didáticos.

---

**Funcionalidades principais**

- Consultar saldo
- Depositar valor
- Transferir valor entre contas
- Encerrar o programa

---

## Tecnologias

- Java (JDK 21)
